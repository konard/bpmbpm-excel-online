# Ограничения Excel Viewer v1.1

## Редактирование файлов

### Поддерживается:
- Редактирование значений в существующих ячейках
- Редактирование текстовых и числовых значений
- Сохранение изменений в формате .xlsx
- Скачивание отредактированного файла

### Ограничения:
1. **Листы**
   - Нельзя добавлять новые листы
   - Нельзя удалять существующие листы
   - Нельзя переименовывать листы

2. **Формулы**
   - При редактировании ячейки с формулой, формула заменяется на введённое значение
   - Нельзя вводить новые формулы (только значения)
   - Формулы подсвечиваются для информации, но не пересчитываются при изменении других ячеек

3. **Форматирование**
   - Не сохраняется форматирование ячеек (цвета, шрифты, границы)
   - Не поддерживаются объединённые ячейки
   - Ширина колонок не сохраняется в файл

4. **Типы данных**
   - Даты отображаются как числа (Excel serial date)
   - Не поддерживаются изображения и диаграммы
   - Не поддерживаются комментарии к ячейкам

5. **Размер файла**
   - Большие файлы (>5 МБ) могут загружаться медленно
   - Очень большие таблицы могут влиять на производительность браузера

## Загрузка файлов по URL

### Ограничения:
- Сервер, на котором размещён файл, должен разрешать CORS-запросы
- Не поддерживаются защищённые паролем файлы
- Не поддерживается авторизация для доступа к файлу

## Поиск и фильтрация

### Поддерживается:
- Поиск по всем колонкам или по выбранной колонке
- Фильтрация ячеек с формулами
- Подсветка найденных совпадений

### Ограничения:
- Поиск только по текущему листу
- Поиск без учёта регистра, но без поддержки регулярных выражений

## Совместимость браузеров

Приложение протестировано и работает в современных браузерах:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

Старые браузеры (IE11 и ниже) не поддерживаются.
